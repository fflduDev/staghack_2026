## dependencies

- python 3.x
    https://www.python.org/downloads/

- pip
    sudo apt install python3-pip

- requests module 
    pip install requests

- flask module
    sudo pip3 install flask --break-system-packages --ignore-installed blinker
    
## usage
to run the program: 
    - run > python app.py 
    - note the port - 9091 by default
    - navigate to browser & request http://localhost:9091/
